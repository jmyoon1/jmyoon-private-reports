Inferring Concept Prerequisite Relations from Online Educational Resources

Sudeshna Roy et al. · IAAI-19 · 2018

arXiv:1811.12640 · PDF · SP: SP3

P0Track A

Problem
해결하려는 문제

Inferring directed prerequisite relations between fine-grained educational concepts (not just courses) at scale. Manual expert annotation does not scale across domains/backgrounds. Prior methods either (a) use hand-tuned semantic features (CGL, MOOC-RF), or (b) solve a quadratic program whose constraint count grows with the number of course-prerequisite edges (CPR-Recover), so they scale poorly and/or need substantial labeled data. Relatedness signals also fail to capture directionality of the dependency.

단순 관련도 임베딩은 개념 유사성만 포착할 뿐 선행 방향성을 표현하지 못한다.

Solution / Contribution
해결책 / 기여

Two-stage supervised pipeline ( on inference details):

  1. Concept representation via Pairwise-Link LDA over documents + document prerequisite graph → asymmetric latent topic vectors per concept (exp-normalized columns of $\beta$).
  2. Directed classification via Siamese network: two tied-weight FC+ReLU sub-networks $G_w$ map $(c_s, c_t)$ to $(v_1, v_2)$; score $f = W^T(G_w(c_s^T) - G_w(c_t^T)) + b$; cross-entropy loss gives $P_{c_s,c_t} = \frac{e^{f_{y_i}}}{\sum_{j\in\{0,1\}} e^{f_j}}$. Negative samples = reversed positive pairs + random unrelated pairs (oversampled to 1.5×) to force learning of directionality.

Notably, PREREQ can substitute video-playlist temporal order for course prerequisite edges (a later video in a playlist is treated as depending on earlier ones), removing the need for manually annotated course dependencies.

pairwise-link LDA와 Siamese 네트워크로 방향성 있는 선행 edge를 추론하며, 토픽 모델 단독이 정밀도 0.98/재현율 0.16임을 절제 실험으로 보여 방향 모델링의 필요성을 입증한다.

Key Results
주요 결과
  • University Course Dataset — PREREQ: Precision 46.76, Recall 91.64, F-score 59.68; beats CPR-Recover and MOOC-RF (exact baseline F-scores truncated in skim).
  • Pairwise-LDA-only baseline: Precision 98.27 but Recall 16.42 → F 16.66 — confirms topics capture relatedness but not directionality; the Siamese stage is what recovers recall/direction.
  • Performance drops only marginally when trained on 40% of labeled data (Fig 7a) — data-efficient.
  • Works on NPTEL MOOC video playlists with no course-prerequisite links, using temporal order as proxy.
Concepts Introduced도입된 개념

PREREQ (the method)

  • What it computes: input = ordered pair of concept latent vectors $(c_s, c_t)$ → Siamese network with cross-entropy loss → output = probability $P_{c_s,c_t}$ that $c_s$ is a prerequisite of $c_t$ (binary directed edge).
  • Why it was created: prior methods (CGL, CPR-Recover, MOOC-RF) rely on hand-tuned semantic/context features and/or a quadratic optimization whose constraints scale with the number of course-prerequisite edges; PREREQ instead learns features automatically and scales to less training data.
  • Concrete example: "Eigen Analysis" → "Principal Components Analysis" is a prerequisite edge in the concept space, inferred from the course-level edge "Linear Algebra → Machine Learning".
  • How it is used in this paper: it is the full pipeline being proposed and evaluated on the University Course and NPTEL MOOC datasets.

Pairwise-Link LDA (as applied here)

  • What it computes: input = document bag-of-words (n-grams, $n\in\{1,2,3\}$) + directed document prerequisite graph $G_D$ → generative topic model with an explicit Bernoulli link variable $e_{d,d'}$ per ordered document pair → output = word-over-topic distribution $\beta$ ($K\times|V|$) and asymmetric topic-topic relation $\alpha$ ($K\times K$); exponentiated/normalized columns of $\beta$ become the $[1\times K]$ concept vectors.
  • Why it was created: borrowed from Nallapati et al. 2008 (citation modeling); needed here because plain relatedness embeddings capture concept similarity but not prerequisite directionality. The asymmetric Bernoulli parameter $\alpha_{z_{dd'},z_{d'd}} \neq \alpha_{z_{d'd},z_{dd'}}$ encodes direction.
  • Concrete example: $K=100$ topics, Dirichlet param $\lambda=0.01$; concept "vector" becomes a 100-dim topic vector.
  • How it is used in this paper: produces the concept representations fed into the Siamese network. Ablation (Fig 7b) shows these beat plain LDA and Wikipedia word2vec.

Concept graph $G_C(C, E_C)$ / document graph $G_D(D, E_D)$

  • What it computes: DAGs where nodes are concepts (or documents) and a directed edge $(c_i, c_j)$ means $c_i$ is a prerequisite of $c_j$. Task = infer unknown concept edges $E_C^U$ from known document graph $G_D$ + labeled concept edges $E_C^T$.
  • Why it was created: formalizes "dependency extraction" as directed-edge prediction over a fixed known concept space.
  • Concrete example: University Course Dataset — 654 course docs, 861 course edges, 1008 labeled concept-pair edges over 365 concepts.
  • How it is used in this paper: the target structure PREREQ predicts; directly relevant to KB/dependency-graph construction (SP3).

5Cs Analysis5Cs 분석

Category: new method (supervised dependency/relation extraction) Context: builds on Pairwise-Link LDA (Nallapati 2008), Siamese nets (Bromley 1994); competes with CGL (Liu 2016), CPR-Recover (Liang 2017), MOOC-RF (Pan 2017). Correctness: results averaged over 5 train-test splits; uses Precision@K (K=50,100) plus P/R/F. No CIs reported in skimmed text. Contributions: (1) PREREQ pipeline (pairwise-link LDA + Siamese) for directed concept prerequisites; (2) ability to learn from unlabeled video playlists via temporal proxy; (3) strong performance with little training data. Clarity: architecture and loss are given explicitly; LDA inference deferred to Nallapati 2008. Source code released. Likely reproducible.

Contribution vs. Claim기여 대 주장

— abstract claims "outperforms state-of-the-art" and "generalize to any domain," but skimmed experiments cover only two CS-domain datasets (University Course + NPTEL MOOC), both using the same 1008 concept pairs. Cross-domain generality is asserted, not demonstrated. Full experiments section needed to confirm the gap.

Implementation Feasibility구현 실현가능성
  • Needs: text corpus + BoW (1/2/3-grams, stopword-removed), a document prerequisite graph or playlist ordering, labeled concept pairs; pairwise-link LDA (mean-field variational inference, $K=100$, $\lambda=0.01$); Siamese net (2 FC+ReLU, tied weights, Adam, lr 1e-4, batch 128, 3500 iters). Modest compute.
  • Code: available at https://github.com/suderoy/PREREQ-IAAI-19/
  • Modifications: for a general KB setting, need an upstream concept-extraction step and a substitute for the document prerequisite graph (temporal, citation, or curricular ordering); retune $K$ and negative-sampling ratio.
  • Unknowns: missing link: how well the fixed-concept-space assumption and the temporal-order proxy transfer to non-educational corpora; exact baseline F-scores and variance (truncated in skim).
Follow-up Papers관련 후속 논문
  • CPR-Recover (Liang et al. 2017) — main baseline; unsupervised QP formulation of the same concept-prerequisite task.
  • MOOC-RF (Pan et al. 2017) — supervised feature-based concept prerequisite recovery from Coursera video transcripts.
  • CGL (Liu et al. 2016) — maps courses onto a universal concept space to predict course+concept prerequisites.
Condition Delta — transfer to our setting
조건 차이 — 우리 환경으로의 전이

Paper's setting: inputs = fixed known concept space $C$, text documents $D$ (course pages / video subtitles), a directed document prerequisite graph $G_D$, and labeled concept-pair edges $E_C^T$; outputs = predicted directed concept prerequisite edges $E_C^U$; conditions = small CS-education benchmarks (~350–650 docs, ~1000 labeled pairs), supervised, concept space assumed fixed and given. Our setting (SP3, dependency extraction for KB construction): inputs = general corpus/entities; outputs = a dependency/prerequisite graph among extracted concepts; conditions = concept set likely not fixed in advance, broader domains, possibly no course-level prerequisite scaffold. Differences:

  • Fixed, pre-known concept space $C$: missing link — our setting may require concept extraction/discovery first, which PREREQ assumes solved.
  • Requires a document-level prerequisite graph $G_D$ (or playlist temporal order) as supervision: reference only — a valuable proxy idea (temporal/ordering signal as weak supervision) but our corpus may lack an analogous ordering.
  • Directed-edge asymmetry via pairwise-link LDA + Siamese: resolved / directly borrowable — the mechanism for capturing dependency directionality is domain-agnostic.
  • Evaluation on 1008 CS concept pairs: reference only — metric (Precision@K, F-score on directed edges) is reusable; benchmark is domain-specific.
