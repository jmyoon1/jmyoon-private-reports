Automatic classification of citation function

Teufel et al. · EMNLP 2006 · 2006 · 512 citations인용 512회

SP: SP1

P0Track C

Problem
해결하려는 문제

Bibliometric impact metrics (and early automatic citation indexers like CiteSeer/ISI) count citations as undifferentiated links, ignoring why a paper is cited — whether it is criticised, used, extended, or merely mentioned in passing. Prior citation-motivation schemes from discourse studies were (a) never validated for inter-annotator reliability, (b) applied only to a few hundred citations by hand, and (c) never automated. There was no reliable, reproducible, machine-applicable scheme for the rhetorical function of a citation.

인용을 단순 카운트로 취급하면 저자가 특정 논문을 왜 인용했는지를 구분하지 못하며, 실질적 방법 의존과 형식적 언급을 나눌 수 없다.

Solution / Contribution
해결책 / 기여

Two-part contribution (empirical + system):

  1. A 12-category annotation scheme for citation function, adapted from Spiegel-Rüsing (1977), with 25-page / ~150-rule guidelines, a decision tree, and a rule that only explicitly signalled functions may be tagged (annotator must cite textual evidence). Shown reliably annotatable.
  2. A supervised classifier (memory-based learning, Weka IBk, k=3, 10-fold CV) using shallow + linguistically-inspired features: cue-phrase grammar (1762 phrases), annotator-contributed cues (892), agent/action recognisers, plus verb tense/voice and sentence-location features.

인용 function(저자의 인용 이유)을 정의하고 12개 범주의 수사적 주석 체계와 최초의 자동 분류기를 제시하여, 이후 citation function 연구의 어휘 표준을 확립한다.

Key Results
주요 결과
  • Human inter-annotator agreement: κ = 0.72 (n=12, N=548, k=3) on the full 12-category scheme.
  • Automatic classification (IBk, 10-fold CV): κ = 0.57, accuracy = 0.77 on the full 12-way scheme.
  • Collapsed 3-way (Weak / Positive / Neutral): κ = 0.58, accuracy = 0.83; precision > 0.75 for all top-level categories.
  • Category distribution is strongly skewed: Neut 62.7%, usage categories (PUse+PModi+PBas) 18.9%, clearly negative only 4.1%.
  • Strong empirical relationship found between citation function and sentiment classification.
Concepts Introduced도입된 개념

Citation function

  • What it computes: input = a citation instance in running text (formal citation or a cited author's name outside a formal citation) → assign exactly one of 12 rhetorical categories describing the author's reason for citing → output = a single categorical label per citation.
  • Why it was created: purely quantitative citation counting treats all citations equally, but many are "politeness, policy or piety" (perfunctory) rather than substantive; prior sociology-of-science schemes were never cross-validated with independent annotators and never automated.
  • Concrete example: category PUse = "author uses cited work's tools/algorithms/data/definitions"; Weak = "weakness of cited approach"; CoCoXY = "contrast between two other cited methods".
  • How it is used in this paper: it is the target label space for both the human annotation study (κ agreement) and the supervised classifier.

The 12-category annotation scheme (Teufel scheme)

  • What it computes: a flat 12-way tagset organized under a top-level 4-way distinction — Weakness (Weak); Contrast/comparison (CoCoGM, CoCo-, CoCoR0, CoCoXY); Positive/agreement/usage (PBas, PUse, PModi, PMot, PSim, PSup); and Neutral (Neut).
  • Why it was created: adapted from Spiegel-Rüsing (1977), redefined to be reliably annotatable (avoids sociological distinctions like "paying homage to pioneers" that need deep field knowledge) while staying informative for citation-indexing / bibliometric applications.
  • Concrete example: distribution is highly skewed — Neut = 62.7%, PUse = 15.8%, negative categories (Weak+CoCo-) only 4.1%.
  • How it is used in this paper: it is the "vocabulary standard" the SP builds on; each citation gets exactly one tag, annotators must point to explicit textual evidence.

Meta-discourse cue phrases / agent–action model

  • What it computes: input = sentence → finite grammar of 1762 cue phrases + 185 POS patterns for 2 agent types (self vs. others) + 20 verb "action" clusters → feature vector signalling rhetorical intent.
  • Why it was created: citation function is signalled by how research is talked about (meta-discourse) rather than object-level content; shallow bag-of-words misses this.
  • Concrete example: continuation-of-ideas verb cluster ("adopt, be based on, be inspired by, build on…") co-occurs with PBas; failing verbs ("be limited to, be incapable of…") signal Weak.
  • How it is used in this paper: primary linguistically-inspired feature family fed to the memory-based (IBk) classifier.

5Cs Analysis5Cs 분석

Category: benchmark/annotation-scheme + system (new task definition + supervised baseline) Context: builds on Spiegel-Rüsing (1977), Moravcsik & Murugesan (1975), and the author's own Argumentative Zoning work (Teufel & Moens 2002); corpus = 360 CL conference papers from the cmp-lg archive. Foundational upstream node for SP1 (citation-function classification lineage → ACL-ARC / Jurgens 2018, ref 3). Correctness: reports κ with n/N/k specified; human κ=.72 vs. chance-corrected machine κ — statistically careful (uses Macro-F + Kappa precisely because accuracy over-rewards the dominant Neut class). Contributions: (1) reliable citation-function scheme; (2) first automatic classifier for it; (3) empirical link between citation function and sentiment classification. Clarity: scheme defined at category level in-text; full reproducibility needs the 25-page guideline document (not in paper). Feature engineering is manual and partly proprietary to the group's toolchain.

Contribution vs. Claim기여 대 주장

Abstract claims the scheme is "reliable" and that supervised ML can "replicate human annotation." Demonstrated: κ=.72 human reliability and κ=.57/.77 machine on one CL corpus (360 papers, one language/domain). Gap to check in deep read: generalization beyond computational-linguistics text is asserted by the application framing (citation indexing broadly) but only evaluated on the cmp-lg corpus; per-category performance on rare classes (Macro-F) is the weak point and needs the results tables to assess.

Implementation Feasibility구현 실현가능성
  • Needs: full-text papers with parsed citations (citation processor, ~94% recognition), a POS tagger, the cue-phrase grammar + verb clusters, Weka (IBk). Annotation requires the 25-page guideline document.
  • Code: not available (2006; group-internal toolchain). PDF obtained from ACL Anthology (W06-1613); no arXiv, no S2 open-access PDF — access limited to the ACL Anthology PDF, read in skim mode (pages 1–5 + results grep).
  • Modifications: for our KB setting, replace hand-engineered features with modern encoders and map the 12 categories onto KB relation types; validate cross-domain.
  • Unknowns: missing link: full per-category Macro-F numbers and the complete feature ablation (need the results tables, §4–5, in a deep read); the exact 25-page annotation guideline rules.
Follow-up Papers관련 후속 논문
  • Measuring the Evolution of a Scientific Field through Citation Frames (Jurgens et al., 2018 — ACL-ARC, ref 3) — directly extends this line into a modern citation-function/frame corpus and classifier; core SP1 successor.
  • The ACL Anthology Network / citation-context corpora — provide the scaled data this scheme was designed to be applied over.
Condition Delta — transfer to our setting
조건 차이 — 우리 환경으로의 전이

Paper's setting: inputs = citations in full-text CL conference papers (XML-converted, citation-parsed); outputs = one of 12 rhetorical function labels per citation; conditions = supervised, English, single narrow domain (computational linguistics), hand-engineered cue/agent/verb features, memory-based learner. Our setting (SP1, dependency extraction for KB construction): inputs = scientific-paper citation contexts; outputs = typed dependency/relation labels between works (e.g., uses / extends / contrasts) for knowledge-base edges; conditions = likely cross-domain, aiming at scalable/automatic extraction feeding a structured KB. Differences:

  • Label space: paper's 12 rhetorical categories vs. our KB relation types — reference only (the scheme is the vocabulary standard we may map onto KB edge types); resolved: top-level 4-way (Weak/Contrast/Positive/Neutral) maps cleanly onto contrast/usage KB edges.
  • Feature approach: hand-built cue grammar (2006) vs. modern neural/LLM extraction — missing link (needs updating; features are pre-neural).
  • Domain: single CL corpus vs. multi-domain KB — missing link (generalization unverified here).
  • Unit: citation instance incl. bare author names vs. entity-pair relation — reference only.
