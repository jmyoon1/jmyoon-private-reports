Conclusion

Build the three-stage pipeline now on the SP6 open-schema-extraction + embedding-threshold link-or-spawn spine; bolt on an external transitive-reduction and cycle-check layer the surveyed literature does not provide.

Answers to Research Questions

결론

지금 SP6의 open-schema 추출 + 임베딩 임계 링크-대-생성 척추 위에 3단계 파이프라인을 구축하고, 조사 문헌이 제공하지 않는 외부 이행 축소·cycle 검사 층을 덧붙인다.

연구 질문에 대한 답변

O1 (P1): Extract prerequisite-concept + citation-lineage signals. Answered.

The literature supplies reliable extraction machinery — citation-intent on the ACL-ARC 6-way schema (~68–71 macro-F1, SciBERT the +12-F1 lever), span-based body-text extraction (~67/~48 SciERC F1), and open-schema LLM extraction as the no-training agent route. But the dependency edge sits one methodized step beyond every benchmark: intent labels are sentence-level, no schema carries a requires type, and the strong extractors are all supervised.

O1 (P1): 선행-개념 + 인용-계보 신호 추출. 충족.

문헌은 신뢰할 추출 기계를 공급한다 — ACL-ARC 6-way 인용 의도(~68–71 macro-F1, SciBERT가 +12 F1 레버), span 기반 본문 추출(~67/~48 SciERC F1), 학습-free 에이전트 경로로서 open-schema LLM 추출. 그러나 의존 edge는 모든 벤치마크에서 한 단계 너머다: 의도 라벨은 문장 수준이고, 어떤 스키마도 requires 유형을 담지 않으며, 강한 추출기는 모두 지도학습이다.

O2 (P1): Match signals to existing nodes or spawn new. Answered, threshold imported not calibrated.

The matching mechanism is solved and agent-adoptable: zero-shot dense linking (BLINK retrieve-then-rerank, SapBERT cosine, GENRE name generation) links unseen nodes without retraining. The reuse-vs-spawn threshold — deferred by academic EL (four of six papers closed-world) — is supplied operationally by the KG-construction toolchain (KARMA ρ, iText2KG θ_E=0.8, Neo4j 0.97+edit-5, EDC verify), and ontology alignment adds the ⟨n,R⟩+threshold primitive with the disputed-mapping new-node signal. The direction must be asymmetric (symmetric similarity P0.98/R0.16). The gap: the threshold is uncalibrated on any scientific dependency KB.

O2 (P1): 신호를 기존 노드에 매칭하거나 신규 생성. 충족, 임계값은 보정이 아닌 수입.

매칭 메커니즘은 해결·에이전트 채택 가능하다: zero-shot 밀집 링킹(BLINK retrieve-then-rerank, SapBERT cosine, GENRE 이름 생성)이 재학습 없이 unseen 노드를 링킹한다. 학술 EL이 미룬 재사용-대-생성 임계값(6개 중 4개 closed-world)은 KG 구성 툴체인이 운용 공급하며(KARMA ρ, iText2KG θ_E=0.8, Neo4j 0.97+edit-5, EDC 검증), 온톨로지 정렬이 disputed-mapping 신규 노드 신호와 함께 ⟨n,R⟩+임계값 primitive를 더한다. 방향은 비대칭이어야 한다(대칭 유사도 P0.98/R0.16). Gap: 임계값이 어떤 과학 의존 KB에서도 미보정이다.

O3 (P2): Keep the growing dependency DAG consistent. Partial — append+prune ✓, cycle handling ✗.

Three clauses, three verdicts. Incremental append is solved (local-graph set-merge, bi-temporal invalidate-not-delete). Redundant-edge pruning is solved exactly by fully-dynamic transitive reduction at O(m) amortized. But cycle detection and resolution is answered by no paper across SP6/SP8/SP9 — agent KG tools enforce no DAG invariants, temporal-KG methods forecast recurring event edges, and hierarchical link prediction either sidesteps cycles or silently contracts SCCs. Because O3 is P2 this does not block convergence, but it is the primary risk in the DAG-maintenance layer.

O3 (P2): 성장하는 의존 DAG를 일관 유지. 부분 — 추가+pruning ✓, cycle 처리 ✗.

세 절, 세 판정. 증분 추가는 해결됨(국소-그래프 set-merge, bi-temporal 무효화-비삭제). 중복-edge pruning은 완전 동적 이행 축소로 O(m) amortized 정확 해결. 그러나 cycle 검출·해소는 SP6/SP8/SP9 전체에서 어떤 논문도 답하지 않는다 — 에이전트 KG 도구는 DAG 불변식을 강제하지 않고, 시계열-KG는 재발 사건 edge를 예측하며, 계층 링크 예측은 cycle을 회피하거나 SCC를 조용히 수축한다. O3가 P2이므로 수렴을 막지 않으나 DAG 유지 층의 주요 위험이다.

Recommended Approach
  1. Stage 1 — Extract (two channels) — open-schema LLM extraction over the body + ACL-ARC citation-intent tagging; add sentence→edge aggregation and an asymmetric prerequisite-typing step. Open-schema extraction is the only no-training route with converging multi-tool support.
  2. Stage 2 — Link-or-spawn — embed each mention, link within θ≈0.8, spawn beyond, gate the borderline band with an LLM equivalence check and the disputed/outKB signal; set direction with the Stage-1 asymmetric check. The threshold rule is converged on by four construction tools.
  3. Stage 3 — Maintain the DAG — incremental set-merge, bi-temporal invalidate-not-delete, fully-dynamic transitive reduction, plus a classical incremental cycle check on every insertion. Cycle detection is answered by no paper, so it is non-optional and sourced classically.
권장 접근법
  1. 1단계 — 추출(두 채널) — 본문 open-schema LLM 추출 + ACL-ARC 인용 의도 태깅; 문장→edge 집계와 비대칭 선행-유형 단계 추가. open-schema 추출은 다중 도구가 수렴 지지하는 유일한 학습-free 경로이다.
  2. 2단계 — 링크-대-생성 — 각 mention을 임베딩해 θ≈0.8 이내면 링크, 초과면 생성, 경계 구간을 LLM 동등성 검사와 disputed/outKB 신호로 게이팅; 1단계 비대칭 검사로 방향 설정. 임계값 규칙은 네 구성 도구가 수렴한다.
  3. 3단계 — DAG 유지 — 증분 set-merge, bi-temporal 무효화-비삭제, 완전 동적 이행 축소, 그리고 모든 삽입에 고전 증분 cycle 검사. cycle 검출은 어떤 논문도 답하지 않으므로 필수이며 고전적으로 조달한다.
Testable Hypothesis

An LLM-agent pipeline combining (i) open-schema extraction + ACL-ARC citation-intent tagging with any-EXTENDS-wins edge aggregation, (ii) embedding-threshold link-or-spawn at θ≈0.8 gated by an LLM equivalence check and an asymmetric directional judgment, and (iii) fully-dynamic transitive reduction plus a classical incremental cycle check, will maintain a globally acyclic, transitively-reduced dependency DAG whose new-node decisions match human judgment above the S2abEL inter-annotator ceiling (~66%) — under a mid-size (10³–10⁴ node) CS/ML KB with LLM-synthesized node descriptions and precision-biased thresholds. Confirmed if reuse-vs-spawn P/R meets ~66% while the cycle check rejects 100% of cycle-closing insertions; refuted if new-node precision falls below S2abEL baselines at every θ.

검증 가능한 가설

(i) any-EXTENDS-wins edge 집계와 함께한 open-schema 추출 + ACL-ARC 인용 의도 태깅, (ii) LLM 동등성 검사와 비대칭 방향 판단으로 게이팅된 θ≈0.8 임베딩 임계 링크-대-생성, (iii) 완전 동적 이행 축소 + 고전 증분 cycle 검사를 결합한 LLM 에이전트 파이프라인은, LLM 합성 노드 설명과 정밀도 편향 임계값을 가진 중규모(10³–10⁴ 노드) CS/ML KB 조건에서, 신규 노드 결정이 S2abEL 주석자 간 상한(~66%)을 상회하는 전역 비순환·이행-축소 의존 DAG를 유지할 것이다. 재사용-대-생성 P/R이 ~66%에 도달하고 cycle 검사가 순환-폐쇄 삽입을 100% 기각하면 확인, 모든 θ에서 신규 노드 정밀도가 S2abEL baseline 이하이면 반증된다.
