Scientific Paper Extractive Summarization Enhanced by Citation Graphs

Xiuying Chen et al. · EMNLP 2022 · 2022 · 14 citations인용 14회

arXiv:2212.04214 · PDF · SP: SP5

P0Track A

Problem
해결하려는 문제

Existing extractive scientific-paper summarization exploits only intra-document structure (section splits, discourse hierarchy, unsupervised graph ranking over sentences). The citation graph — which links a paper to related work sharing domain terms — carries document-level relatedness that had not been leveraged for extractive summarization. Prior citation-context methods also fail when a paper has not yet been cited.

라벨 없는 문서 수준 관련도 신호를 얻기 어렵다.

Solution / Contribution
해결책 / 기여

Two models leveraging the citation graph (see Concepts). MUS (unsupervised): SciBERT finetuned by contrastive link prediction, then multi-granularity position-aware centrality selects sentences. GSS (supervised): GNN encoder + gated sentence encoder + graph information fusion + multi-task (extraction + link prediction) framework.

cited/uncited 삼중항의 contrastive link-prediction으로 학습된 관련도 신호를 제공하여 라벨-free 의존 후보 순위화를 가능케 하며 인용 이웃이 풍부할수록 이득이 커짐을 보인다.

Key Results
주요 결과
  • GSS sets new supervised SOTA on SSN (inductive & transductive) over BertSumEXT, MGSum-ext, EMSum, CGSum, HSG on RG-L and BERTScore.
  • MUS beats unsupervised baselines LEAD, PACSUM, HipoRank.
  • Ablation: graph encoder gives +1.12 RG-L, +0.74 BERTScore on SSN (transductive); removing multi-granularity levels drops RG-L.
  • MUS/GSS improve as the number of cited papers per target paper grows, while HipoRank/HSG stay flat — model benefits from richer citation neighbourhoods.
Concepts Introduced도입된 개념

Citation graph (as summarization signal)

  • What it computes: nodes = papers, edges = citation relationships → the graph encodes document-level relatedness (adjacent papers share domain-specific terms/topics) → used as an inter-document context source beyond a paper's own text.
  • Why it was created: prior extractive scientific-summarization work used only intra-document relationships (sections, discourse hierarchy); the effect of the citation graph on extractive summarization was unexplored.
  • Concrete example: For a target summarization paper, directly cited papers 1–2 and indirectly cited paper 3 share domain terms and are relevant; papers 4–5 (far in the graph, e.g. POS-tagging HMM) are irrelevant.
  • How it is used in this paper: Core input signal for both MUS and GSS; drives link-prediction finetuning and the inter-document sentence centrality.

MUS — Multi-granularity Unsupervised Summarization

  • What it computes: (1) finetune SciBERT on the citation graph via link prediction (triplet: query paper $d$, positive/cited $d^+$, negative/uncited $d^-$; contrastive loss $L = -\log \text{sim}(d,d^+) - \log[1-\text{sim}(d,d^-)]$); (2) build a multi-granularity graph and score sentences by position-aware centrality at sentence-, section-, and document-level → extract top sentences as the abstract.
  • Why it was created: low-cost solution needing no labeled data, to test whether citation-graph structure helps even without supervision.
  • Concrete example: sentence local importance $a_i^{sen} = \sum_j e_{ij}^{sen}$ with edge weight scaled by boundary position (start/end sentences weighted higher, $\lambda_1<\lambda_2$).
  • How it is used in this paper: The unsupervised model; establishes that citation graph helps before building the supervised GSS.

GSS — Graph-based Supervised Summarization

  • What it computes: GNN encoder over pretrained LM produces sentence reps (target doc) and document reps (reference papers) → gated sentence encoder polishes sentence reps by relatedness to document gist → graph information fusion module injects reference-paper info → multi-task head jointly scores sentence extraction and predicts citation edges.
  • Why it was created: to exploit the graph more strongly when large-scale labeled data are available, going beyond unsupervised centrality.
  • Concrete example: graph encoder alone improves GSS by +1.12 RG-L and +0.74 BERTScore on SSN (transductive) in ablation.
  • How it is used in this paper: The supervised SOTA model; main empirical contribution.

5Cs Analysis5Cs 분석

Category: new method (two models: one unsupervised, one supervised) Context: builds on SciBERT (Beltagy 2019), graph-ranking summarization (Mihalcea & Tarau 2004; HipoRank Dong 2021), BertSumEXT (Liu & Lapata 2019), and the SSN dataset / CGSum (An et al. 2021, abstractive citation-graph summarization). Directly relevant to SP5 (citation-graph signals for a paper's derivation neighbourhood). Correctness: ROUGE scores reported with 95% CI ≤ ±0.23; human eval with kappa 0.42–0.51; link-prediction repeated 10× with std. Appears rigorous. Contributions: (1) demonstrates citation-graph modeling helps extractive scientific summarization; (2) MUS + GSS models; (3) new unsupervised and supervised SOTA on SSN with released code. Clarity: equations and module descriptions present in first 5 pages; likely reproducible.

Contribution vs. Claim기여 대 주장

Abstract claims "substantial improvements over prior SOTA" for both unsupervised and supervised settings on a public benchmark. Experiments (SSN inductive + transductive, ROUGE + BERTScore + QuestEval + human eval) appear to support this, but exact margin verification and the generality-beyond-SSN question require reading the full experiments section. Only one dataset family (SSN) is used, so cross-domain generality is claimed intuitively (via the number-of-cited-papers analysis) but not tested on a second corpus.

Implementation Feasibility구현 실현가능성
  • Needs: SSN dataset (An et al. 2021), SciBERT, a GNN/graph encoder, contrastive link-prediction training; GPU for GSS. Multi-granularity graph construction over sections.
  • Code: available — authors state "We release our code" (arXiv:2212.04214).
  • Modifications: for KB dependency extraction, retarget the extraction head to a dependency/relevance-ranking head; MUS's link-prediction finetuning is directly usable as an unsupervised paper-relatedness scorer.
  • Unknowns: missing link — exact ROUGE table numbers, GNN architecture details, and code repo URL not captured in skim; missing link — whether the method degrades on sparse/new-paper citation neighbourhoods (cold-start).
Follow-up Papers관련 후속 논문
  • CGSum / SSN dataset (An et al., 2021) — the abstractive citation-graph baseline and dataset this paper extends; prerequisite reading for SP5.
  • HipoRank (Dong et al., 2021) — hierarchical/positional unsupervised graph baseline that MUS outperforms; relevant intra-document graph alternative.
Condition Delta — transfer to our setting
조건 차이 — 우리 환경으로의 전이

Paper's setting: inputs = scientific papers (sentences) + citation graph over the whole corpus (SSN: ~128K training nodes); outputs = extractive summary (abstract sentences) via per-sentence extraction scores; conditions = requires a corpus-wide citation graph, SciBERT embeddings, and (for GSS) gold extraction labels. Our setting (SP5, dependency extraction for KB construction): inputs = papers and their citation/derivation relationships; outputs = a paper's derivation neighbourhood / dependency structure surfaced from citation-graph signals; conditions = focus on structural relatedness among papers, not abstract-sentence extraction. Differences:

  • Task target (extractive summary vs. dependency/derivation neighbourhood surfacing): reference only — the summary is a downstream use, but the link-prediction finetuning and centrality signals are the reusable core.
  • Citation-graph link-prediction as a relatedness signal: resolved / directly transferable — the contrastive triplet objective ranks cited vs. uncited papers, which is exactly a dependency-relevance signal.
  • Multi-granularity centrality (sentence/section/document): reference only — useful if SP5 needs sentence-level derivation evidence, else document-level suffices.
  • Supervision: GSS needs gold labels; missing link — SP5 may lack labeled dependency data, making MUS (unsupervised) the more transferable half.
